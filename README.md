# Origin Forme Dex

A personal Pokemon collection tracker built as a portfolio project and for personal use.

Track your progress across each Pokemon game, with support for alternate formes, regional variants, gender differences, event Pokémon, and shiny collections. Designed around an "Origin Dex" concept — every Pokémon sourced from its earliest available generation.

## Features planned
- Browse the full Pokemon catalogue by game, version group, or as the full dex
- Track individual mons with notes, tags, nicknames and source game
- Origin Dex % and General Dex % completion tracking
- Shiny and Alpha colleciton tracking
- Local-first with IndexedDB storage, manual export/import for cross-device progression sync

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS + shadcn/vi
- Dexie.js (IndexedDB)
- TanStack Query
- React Router v6
- Data sourced from PokeAPI
